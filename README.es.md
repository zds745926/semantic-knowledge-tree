# Árbol de Conocimiento Semántico (Semantic Knowledge Tree)

> En lugar de memorizar el conocimiento en los pesos del modelo, SKT organiza el conocimiento como un árbol con vectores semánticos. Modelos pequeños de 1.5B pueden superar a modelos de 100B+ recuperando conocimiento a través del árbol.
