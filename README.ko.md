시맨틱 지식 트리(SKT) — 지식을 모델 가중치에 기억시키는 대신, SKT는 지식을 의미 벡터를 가진 트리로 구성합니다. 1.5B의 작은 모델이 트리를 통해 지식을 검색하여 100B+ 모델을 능가할 수 있습니다.
