# Árvore de Conhecimento Semântico (Semantic Knowledge Tree)

> Em vez de memorizar conhecimento nos pesos do modelo, SKT organiza o conhecimento como uma árvore com vetores semânticos. Modelos pequenos de 1.5B podem superar modelos de 100B+ recuperando conhecimento através da árvore.
