# Arbre de Connaissances Sémantiques (Semantic Knowledge Tree)

> Au lieu de mémoriser les connaissances dans les poids du modèle, SKT organise les connaissances sous forme d'arbre avec des vecteurs sémantiques. De petits modèles de 1.5B peuvent surpasser des modèles de 100B+ en récupérant des connaissances via l'arbre.
