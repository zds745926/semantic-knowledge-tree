# Pohon Pengetahuan Semantik (Semantic Knowledge Tree)

> Alih-alih menyimpan pengetahuan di dalam bobot model, SKT mengorganisir pengetahuan sebagai pohon dengan vektor semantik. Model kecil 1.5B dapat melampaui model 100B+ dengan mengambil pengetahuan melalui pohon.

Pohon Pengetahuan Semantik (SKT) — Alih-alih menyimpan pengetahuan di dalam bobot model, SKT mengorganisir pengetahuan sebagai pohon dengan vektor semantik. Model kecil 1.5B dapat melampaui model 100B+ dengan mengambil pengetahuan melalui pohon.
