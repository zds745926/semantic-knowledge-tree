Semantic Knowledge Tree (SKT) — 知識をモデルの重みに記憶させる代わりに、SKTは知識をセマンティックベクトルを持つツリーとして整理します。1.5Bの小さなモデルが、ツリーを通じて知識を検索することで100B+のモデルを凌駕できます。
